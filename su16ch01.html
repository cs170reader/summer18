CS 170 Introduction to Java Programming

Writing Simple Programs


Welcome to CS 170. Before you learn how to program in Java you'll have to spend some time getting prepared. In this first chapter, you'll cover some background material, set up your Java development environment, and write and run your first Java programs.

In this chapter you'll learn:

  • how your computer's memory stores data.
  • how your computer's CPU converts data to information and instructions.
  • the mechanics of Java programming.
  • the basic syntax of a Java console program.
  • how to use Java's console output objects.
  • how to print text output using the String type.
